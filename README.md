# 📊 Customer Segmentation & Retention Analytics

## 🎯 Objective
To create a Power BI dashboard that segments customers based on purchasing behavior and analyzes retention rates, helping businesses design loyalty programs and reduce churn.

## 🛠️ Tools Used
* **Power BI Desktop:** Data visualization, Data Modeling (Star Schema).
* **Power Query:** Data cleaning, formatting, and transformation.
* **DAX:** Creating custom measures and KPIs (CLV, Retention Rate, Churn Rate).

## 💡 Key Business Insights
1. **Revenue Driver:** The 'High Value' segment makes up a small percentage of users but drives the majority of total revenue.
2. **Churn Risk:** Customers who haven't purchased in >180 days are flagged as 'Churned', highlighting a need for a targeted win-back email campaign.
